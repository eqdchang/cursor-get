Drop-in Header Demo

This host page only includes dist/byrna/header.bundle.js. The bundle auto-mounts into the #byrna-header-root div above. Tailwind styles are injected into the document head by the bundle itself — no CSS file required on the host.

You can also mount programmatically:

window.ByrnaHeader.mount("#my-header-slot");
window.ByrnaHeader.unmount();
