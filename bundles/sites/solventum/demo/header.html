Drop-in Header Demo

This host page only includes dist/solventum/header.bundle.js. The bundle auto-mounts into the #solventum-header-root div above. Tailwind styles are injected into the document head by the bundle itself — no CSS file required on the host.

You can also mount programmatically:

window.SolventumHeader.mount("#my-header-slot");
window.SolventumHeader.unmount();
